## Sobre
Olá! Sou **Pedro Lins**, um desenvolvedor front-end de 17 anos com paixão por tecnologia. Este portfólio vanilla HTML/CSS/JS showcase meus projetos, habilidades e jornada. Totalmente **responsivo** e **otimizado** para todas as telas! 🌟

- ✨ Design dark/space theme com gradients e glassmorphism
- 📱 100% Responsivo (mobile-first)
- ⚡ Animações suaves e hover effects
- 🔄 Tabs interativos para projetos
- 🎯 Timeline de experiência

- 🚀 Performance otimizada (sem frameworks)
